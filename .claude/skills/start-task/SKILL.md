name: start-task
description: Starts the next big task or Milestone according to project history. Use when starting with phase 1 of a new milestone or when the user says "start next task" or "proceed with next milestone". 
---

When starting the task, always read - think - propose:

1. Make sure to thoroughly ready through the recent worklogs, memory and specification before starting with anything.
2. Ask the user about things that remain unclear.
3. Invoke subagents at least in stage 3 for review (@.claude/agents/plan-reviewer) and stage 5 for documentation (@.claude/agents/doc-auditor) but also whenever it makes sense.
4. Create the worklog file and keep documenting your thoughts, errors, and findings constantly along the process in order to avoid the errors more than once.
