## 1. research

**Goal:** Replace proprietary JWT login API with OIDC IdP using Spring Authorization Server.

**Current state:** M1 auth service complete with jjwt-based JWT issuance, refresh token rotation, JWKS endpoint. 40/40 unit tests green.

**Key findings:**
- Spring Boot 4.0.5 bundles Spring Security 7.0.4 and Spring Authorization Server 7.0.4
- SS7 API change: `OAuth2AuthorizationServerConfigurer.authorizationServer()` static method removed; use `http.oauth2AuthorizationServer(...)` instead
- `AntPathRequestMatcher` removed in SS7; use `PathPatternRequestMatcher.pathPattern()`
- `OAuth2AuthorizationServerConfiguration.jwtDecoder()` exists in SS7 at `org.springframework.security.config.annotation.web.configuration` (not the old package)
